Features

Everything You Need for Seamless Scripture Display

Divine Link detects Bible verses mentioned during preaching and lets you push them to your presentation software with one click.

Live Speech Recognition

Advanced AI transcription captures every word spoken, detecting scripture references in real-time with high accuracy.

Intelligent Detection

Recognises multiple formats: "John 3:16", "John chapter 3 verse 16", spoken numbers, ranges, and contextual references.

ProPresenter Integration

Push to Stage Display or Audience Screen with one click. Premium users get direct API integration for faster, more reliable display.

Direct API Integration (Premium)

Messages API integration sends scriptures directly to ProPresenter without keyboard simulation. Faster, more reliable, works in background.

Multiple Translations

Access KJV, ESV, NIV, NASB, NLT, and more. Switch between translations to match your church's preference.

Multi-Verse Handling

Detect ranges like "John 3:16-21" and navigate through verses individually. Push one at a time or all at once.

Pastor Profiles

Save preferred translations and speech corrections for each speaker. Perfect for churches with multiple ministers. Grace: 2 profiles, Love: 5 profiles

Secure & Reliable

Airtight authentication with reactive state management. Your subscription, admin access, and premium features are always in sync — no stale data, no leaky permissions.

In-App Support

Premium subscribers can reach our support team directly from within the app via the Contact Us form. Grace and Love tier customers receive dedicated email and priority support.
